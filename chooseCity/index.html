城市选择 #

提供按城市和省份选择中国所有的城市，支持搜索和跳转

基本用法 #

示例见 https://lingdeqingyuan.github.io/element-components/#/chooseCity

代码示例 #

示例见 https://lingdeqingyuan.github.io/element-components/#/chooseCity

事件 #

事件名 说明 参数
changeCity 按城市选择时触发 val: City
changeProvince 按省份选择时触发 val: string

City 数据格式 #

key 说明 类型
id 城市编码 number
spell 城市拼音 string
name 城市名字 string
