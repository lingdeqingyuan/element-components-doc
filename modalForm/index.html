弹出框表单 #

可配置弹出框表单，支持多种自定义。

基本示例 #

示例见 https://lingdeqingyuan.github.io/element-components/#/modalForm

代码示例 #

<modalForm
  v-model:visible="visible"
  title="编辑用户"
  width="50%"
  :options="options"
>
  <template #footer="{ form }">
    <el-button @click="cancel(form)">取消</el-button>
    <el-button type="primary" @click="confirm(form)">确认</el-button>
  </template>
</modalForm>

Props #

属性说明类型可选值默认值
isScroll是否只在可视区域滚动booleantrue / falsefalse
visible弹框显示booleantrue / falsefalse
options表单配置项FormOptions[]必填up
httpRequest上传自定义方法function--

FormOptions 数据格式 #

key说明类型
type表单项类型string
value表单项初始值any
label表单项 labelstring
prop表单项标识string
rules表单项的校验规则同 ElementPlus 校验规则
placeholder表单项占位符string
attrs表单项的属性同 ElementPlus 表单组件属性名字
children表单项的子元素FormOptions[]
uploadAttrs上传选项的配置同 ElementPlus Upload 组件属性名字

事件 #

见 ElementPlus Upload 组件事件 https://element-plus.gitee.io/zh-CN/component/upload.html#upload-属性
