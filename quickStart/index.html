快速开始 #

因为我们当前组件库是基于ElementPlus二次封装的，所有在项目当中必须确保已经安装和正确使用了ElementPlus。

使用指南 #

  1. 在项目目录里安装 npm 包ryuji-element-components

    (1) npm 安装： npm i -S ryuji-element-components

    (2) yarn 安装：yarn add -S ryuji-element-components

  2. 在 main.ts 中引入

import UI from "ryuji-element-components";
import "ryuji-element-components/style.css";
  1. 按需引入
import ChooseIcon from "ryuji-element-components/ChooseIcon";
import "ryuji-element-components/style.css";
