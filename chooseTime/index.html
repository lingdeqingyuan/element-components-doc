时间选择器 #

提供开始时间与结束时间选择联动。

示例见 https://lingdeqingyuan.github.io/element-components/#/chooseTime

代码示例 #

<z-chooseTime></z-chooseTime>

Api #


Props #

属性说明类型可选值默认值
startPlaceholder开始时间占位符string-请选择开始时间
endPlaceholder结束时间占位符string-请选择结束时间
startTimeStart开始时间的开始选择string-08:00
startStep开始时间的步进string-00:30
startTimeEnd开始时间的结束选择string-24:00
endTimeStart结束时间的开始选择string-08:00
endStep结束时间的步进string-00:30
endTimeEnd结束时间的结束选择string-24:00

事件 #

事件名说明参数
startChange开始时间变化时触发val:string
endChange结束时间变化时触发val:Value

Value 数据格式 #

key说明类型
startTime开始时间string
endTime结束时间number
趋势标记
城市选择
