省市区选择 #

通过下拉选择实现省市区三级联动

基本用法 #

示例见 https://lingdeqingyuan.github.io/element-components/#/chooseArea

代码示例 #

<choose-area></choose-area>

Api #


事件 #

事件名 说明 参数
change 选择了省市区三级之后触发 val:AreaInfo, 省市区数据

AreaInfo 数据格式 #

key 说明 类型
province 省份数据 val: Value
city 城市数据 val: Value
area 区域数据 val: Value

Value 数据格式 #

key 说明 类型
name 名称 string
code 编码 number
