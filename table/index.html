表格 #

可配置型表格，支持自定义

基本用法 #

示例见 https://lingdeqingyuan.github.io/element-components/#/table

代码示例 #

<z-table
  :tableData="data"
  :tableOption="options"
  :currentPage="currentPage"
  :pageSize="pageSize"
  :total="total"
  border
  stripe
  :editIndex="editIndex"
  @confirm="confirm"
  @cancel="cancel"
  @size-change="handleSizeChange"
  @current-change="handleCurrentChange"
>
  <template #date="{ scope }">
    <icon-timer></icon-timer>
    <span>{{ scope.row.date }}</span>
  </template>
  <template #name="{ scope }">
    <el-popover effect="light" trigger="hover" placement="top" width="auto">
      <template #default>
        <div>name: {{ scope.row.name }}</div>
        <div>address: {{ scope.row.address }}</div>
      </template>
      <template #reference>
        <el-tag>{{ scope.row.name }}</el-tag>
      </template>
    </el-popover>
  </template>
  <template #editRow="{ scope }">
    <el-button
      size="small"
      @click="handleEdit(scope.$index, scope.row)"
      type="primary"
      >完成</el-button
    >
    <el-button
      size="small"
      type="danger"
      @click="handleDelete(scope.$index, scope.row)"
      >取消</el-button
    >
  </template>
  <template #action="{ scope }">
    <el-button
      size="small"
      @click="handleEdit(scope.$index, scope.row)"
      type="primary"
      >编辑</el-button
    >
  </template>
</z-table>

Api #


Props #

属性说明类型可选值默认值
tableOption表格配置项TableOption[]必填-
tableData表格数据any[]必填-
elementLoadingText加载文案string--
elementLoadingBackground加载背景颜色string--
elementLoadingSpinner加载图标名string--
elementLoadingSvg加载图标 svgstring--
elementLoadingSvgViewBox加载图标 svg 的配置string--
editIcon编辑显示的图标string-'Edit'
isEditRow是否可以编辑行booleantrue / falsefalse
pagination是否显示分页booleantrue / falsefalse
paginationAlign分页对齐方式stringleft / right / centerleft
currentPage当前是第几页number-1
pageSize一页多少数据number-10
pageSizes一页可选择的数据条数number[]-[10, 20, 30, 40]
total数据总数number--

TableOption 数据格式 #

key说明类型
prop字段名string
label表头string
width列宽string / number
align对齐方式string
slot自定义列模板的插槽名string
action是否是编辑选项boolean
editable单元格是否可编辑boolean

事件 #

事件名说明参数
confirm可编辑单元格确认事件scope: 当前行数据
cancel可编辑单元格确认事件scope: 当前行数据
size-change当前页数据条数改变事件val: 当前页条数
current-change当前页改变事件val: 当前页条数

slots 插槽 #

名称说明
cellEdit自定义可编辑单元格内容
editRow自定义可编辑行按钮内容
action自定义编辑区域内容
